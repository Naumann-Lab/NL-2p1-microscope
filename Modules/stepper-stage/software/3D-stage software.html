3D-stage software

This software was designed to provide basic control of stages from different manufacturers, and currently allows using the stages controlled by the firmware described in 3D stage firmware, as well as Newport and Thorlabs linear stages. It is also designed as the MC module, allowing for integration into MC assemblies.

The front panel of the 3D-stage module is made as compact as possible to allow for efficient integration into large MC assemblies. All controls and indicators of each motor axis which are not used routinely, are moved into the settings windows. General stage and joystick controls are set separately in the common settings (gear wheel outside of the tab window).

The changing the target control moves the axis of the stage corresponding to the selected tab. The units of the target are set in the settings window. When the stage is moving, the moving indicator is lit. Choosing one of the presets will automatically move the stage to the corresponding position. Be careful, as some of the positions might interfere with the current sample placement. Pressing the red button stops the stage via deceleration (and it acceleration is set to a low value in the settings, it might take a while). joy push button toggles the joystick control for all axes.

Intended future updates

  • At the moment, the controller cannot be pre-programmed with a series of commands. All of the commands are executed as they are received sequentially from a computer. However, this option will likely appear in future updates.
  • At the moment, the controller operates in an open loop configuration, as there is no encoder for the current position of the stage. In the future, it will likely change.
